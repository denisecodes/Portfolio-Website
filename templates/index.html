About

I am a former Tech Recruiter and HR professional, looking to transition into a data role. Passionate data individual, seeking to use proven skills in SQL, Python and Tableau to analyse data for a fast-growing scale-up/start-up. Completed 60 python course projects, 1 SQL database on Badminton Players, 1 Tableau Dashboard on London Housing and contributed to 1 python recipe search project.

I started to learn to code due to my curiosity to understand what tech people do and in the process, fell in love with SQL and Python. Currently, I am learning SQL, Python and Tableau. I have completed two courses with Code First Girls (CFG): Intro to Data and SQL and Intro to Python and Apps, two courses on Udemy: “The Complete SQL Bootcamp: Go from Zero to Hero" and “Tableau 2022 A-Z: Hands-On Tableau Training for Data Science” and Alex the Analyst: Data Analyst Bootcamp.

MySQL

Badminton Players

GitHub link
This is a solo project focused on badminton players information and match statistics. I created SQL queries to analyse if factors such as gender, age and country has a trending pattern in the ranking of badminton players. For the analysis, I used view, stored function, stored procedure and joins.

Pokémon Analysis

GitHub link
This is a personal project I did to analyzed the Pokémon dataset on Kaggle. In this project, I analyzed the dataset to find out how many Pokémon is in each generation, which generation has the strongest and weakest Pokémon, what Pokémon type is the strongest and the weakest.

Python

Recipe Search

GitHub link
This is a group project completed for Code First Girls Kickstarter Python course. This project helps users to find recipes to cook based on certain parameters: meal type, cuisine type and ingredient. For this code, we used an API to retrieve recipes.

Automate Booking on Spond

GitHub link
This is a personal project that I made to automate booking my badminton sessions on Spond due to high popularity and competitiveness to get a spot. I plan to host this code on PythonAnywhere so it can run there automatically every two weeks to help me book sessions without me doing so.

Tableau

London Housing

GitHub link
This is a personal project I did to analyze the relationship between housing prices, houses sold, salary and crime between 1995 to 2020 in London. I created a Dashboard with 5 different visualizations including a map, line graphs, numbers and filters for users to quickly understand the housing situation in London over the last two decades. You can find in my READ.md file my findings and analysis.
